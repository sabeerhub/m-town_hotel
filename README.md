# M-Town Hotel

A modern, responsive hotel website showcasing elegant design and user-friendly interface for browsing accommodations, amenities, and booking services.

## Overview

M-Town Hotel is a professional web-based platform designed to provide an exceptional digital experience for hotel guests and visitors. The website features a clean, contemporary design combined with smooth, interactive elements to showcase the hotel's services and facilitate customer engagement.

## Features

- **Responsive Design**: Fully responsive layout that adapts seamlessly to desktop, tablet, and mobile devices
- **Modern UI/UX**: Professional and elegant interface designed for optimal user experience
- **Interactive Elements**: Smooth animations and interactive features powered by JavaScript
- **Hotel Information**: Comprehensive sections for rooms, amenities, services, and facilities
- **Intuitive Navigation**: Easy-to-use navigation menu for seamless browsing
- **Contact & Booking**: Accessible contact information and booking inquiry sections

## Technology Stack

- **HTML** (49.2%) - Semantic markup and structured content
- **CSS** (43.6%) - Styling, layout, and responsive design
- **JavaScript** (7.2%) - Interactive features and dynamic functionality

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No additional installations required

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sabeerhub/m-town_hotel.git
   ```

2. Navigate to the project directory:
   ```bash
   cd m-town_hotel
   ```

3. Open the website:
   - Simply open `index.html` in your web browser, or
   - Use a local server for optimal experience:
     ```bash
     python -m http.server 8000
     # or
     npx http-server
     ```

4. Visit `http://localhost:8000` in your browser

## Usage

The website is ready to use immediately after opening. Navigate through the different sections using the menu to explore:
- Hotel information and facilities
- Available rooms and rates
- Amenities and services
- Contact and booking options

## Key Sections

- **Home**: Welcome page with featured highlights
- **Rooms**: Detailed room descriptions and specifications
- **Amenities**: List of hotel facilities and services
- **Gallery**: Visual showcase of hotel spaces
- **Contact**: Guest inquiry and reservation options

The website is optimized for:
- Desktop (1024px and above)
- Tablet (768px - 1023px)
